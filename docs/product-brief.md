# GAIA 제품 기획서

## 한 줄 설명

`gaia`는 지구 곳곳에서 발생하는 자연 파괴 이벤트를 제한 시간 안에 직접 조작 미션으로 해결하는 웹 기반 실시간 미니 게임입니다.

## 핵심 경험

사용자는 io 게임처럼 이름만 입력하고 바로 시작합니다. 게임이 시작되면 화면 중앙에 회전하는 지구가 보이고, 지구 위 여러 위치에 이벤트 포인트가 생성됩니다. 포인트는 시간이 지날수록 노랑, 주황, 빨강, 검정으로 위험도가 올라갑니다. 검정이 되기 전에 해당 위치로 들어가 문제를 해결해야 하며, 검정이 되면 그 이벤트는 실패 처리되고 사라집니다.

사용자가 지구 위 이벤트 포인트를 누르면 지구 화면은 그대로 유지된 상태에서 선택 지점을 중심으로 현장 장면이 확대되어 전체 화면을 채웁니다. 이벤트 페이지에는 산불, 해양 쓰레기, 기름 유출 등 자연을 해치고 있는 상황을 이벤트별 미니게임으로 표시합니다. 사용자는 별도 스킬을 고르지 않고 현장 안의 오브젝트를 직접 드래그하거나 클릭해 문제를 해결합니다. 예를 들어 해양 쓰레기 이벤트는 바다와 모래사장 위에 흩어진 쓰레기를 주워 수거함에 버리면 완료됩니다. 미션을 정확히 끝내면 실제 환경 정보 결과 화면이 나타나고, 사용자가 확인하면 해당 해결 기록이 업적으로 추가됩니다.

정해진 개수의 이벤트가 끝나면 게임이 종료됩니다. 마지막 화면에서는 획득한 업적, 끝낸 시간, 검정이 되기 전에 해결한 이벤트, 검정이 되어 해결하지 못한 이벤트를 종합해 점수를 계산하고 랭킹에 저장합니다.

## 주요 화면

1. 시작 화면
   - 플레이어 이름 입력
   - 시작 버튼
   - 간단한 게임 제목 `GAIA`

2. 지구 상태 화면
   - 중앙의 회전하는 지구
   - 지구 위 이벤트 포인트
   - 이벤트 위험도 색상 표시
   - 우측 상단에서 아래로 나열되는 업적 목록
   - 남은 이벤트 수, 경과 시간, 점수 표시

3. 이벤트 해결 화면
   - 지구 위 선택 좌표에서 현장 장면으로 확대되는 진입 연출
   - 이벤트별 직접 조작 미니게임
   - 현재 이벤트 위험도
   - 미션 목표와 작업 진행률
   - 현장 오브젝트를 드래그하거나 클릭하는 직접 조작
   - 미션 완료 시 결과 정보 표시
   - 우측 업적 목록 유지

4. 결과 정보 화면
   - 어떤 문제를 해결했는지 표시
   - 실제 환경 문제 설명
   - 관련 법, 정책, 뉴스, 참고 정보 표시
   - 확인 버튼

5. 게임 종료 화면
   - 최종 점수
   - 완료 시간
   - 해결한 이벤트 수
   - 실패한 이벤트 수
   - 획득 업적
   - 랭킹 저장 결과

## 플레이 목표

- 제한 시간 안에 가능한 많은 이벤트를 직접 조작 미션으로 해결합니다.
- 자연 파괴 문제에 대한 실제 정보를 게임 안에서 짧게 학습합니다.
- 플레이 기록을 랭킹에 남겨 다시 도전할 동기를 만듭니다.

## 초기 개발 범위

- 이름 입력 후 게임 시작
- 회전하는 지구 화면
- 이벤트 포인트 생성과 위험도 변화
- 이벤트 상세 화면 이동
- 이벤트별 미니게임과 완료 판정
- 해결 결과와 업적 기록
- 게임 종료와 점수 계산
- Supabase 랭킹 저장
