# GAIA 게임 디자인

## 게임 루프

1. 사용자가 이름을 입력하고 게임을 시작합니다.
2. 지구 화면에 자연 파괴 이벤트 포인트가 생성됩니다.
3. 각 이벤트는 시간이 지날수록 위험도가 상승합니다.
4. 사용자가 이벤트 포인트를 클릭하면 지구를 유지한 채 선택 위치에서 현장 장면이 확대되어 이벤트 해결 화면으로 이어집니다.
5. 사용자가 이벤트별 직접 조작 미션을 수행합니다.
6. 미션 성공 조건을 만족하면 문제 해결 결과 화면이 열리고 업적이 추가됩니다.
7. 이벤트가 검정 단계까지 도달하면 실패 처리되고 사라집니다.
8. 목표 이벤트 수가 모두 해결 또는 실패되면 게임이 종료됩니다.
9. 최종 점수를 계산하고 랭킹에 저장합니다.

## 이벤트 위험도

이벤트 포인트는 아래 단계를 가집니다.

| 단계 | 색상 | 의미 |
| --- | --- | --- |
| 1 | 노랑 | 초기 발생 |
| 2 | 주황 | 확산 중 |
| 3 | 빨강 | 긴급 상황 |
| 4 | 검정 | 복구 실패, 이벤트 소멸 |

초기값은 각 이벤트마다 동일하게 둘 수 있지만, 이후 난이도에 따라 단계별 지속 시간을 다르게 설정할 수 있습니다.

## 초기 이벤트 목록

| 이벤트 ID | 상황 | 미션 방식 | 해결 설명 | 실패 설명 |
| --- | --- | --- | --- | --- |
| wildfire | 산불 | 준비 중 | 불길을 직접 진압합니다. | 숲이 불타 생태계가 파괴됩니다. |
| ocean_trash | 바다 쓰레기 | 쓰레기 드래그 수거 | 바다와 모래사장에 흩어진 쓰레기를 수거함에 버립니다. | 해양 생물이 플라스틱을 삼키거나 얽힙니다. |
| oil_spill | 기름 유출 | 준비 중 | 오염된 바다를 회복합니다. | 오염이 퍼져 해양 생태계가 손상됩니다. |
| illegal_logging | 불법 벌목 | 준비 중 | 훼손된 벌목지를 복원합니다. | 숲이 사라져 서식지가 줄어듭니다. |
| air_pollution | 대기 오염 | 준비 중 | 오염된 공기를 정화합니다. | 미세먼지와 유해가스가 확산됩니다. |
| drought | 가뭄 | 준비 중 | 마른 땅과 작물을 회복합니다. | 토양이 메마르고 농작물이 죽습니다. |

## 이벤트 진입과 현장 표현

- 이벤트 포인트 클릭 시 해당 포인트 위치를 확대 전환의 중심으로 사용합니다.
- 확대 중에는 회전 지구가 배경에 유지되고, 현장 미니게임이 화면을 덮습니다.
- 이벤트 상황은 단순 이미지 선택이 아니라 클릭, 드래그, 정렬, 타이밍 입력 같은 직접 조작으로 해결합니다.
- 해양 쓰레기 미션은 바다와 모래사장에 배치된 쓰레기 오브젝트를 수거함까지 드래그하면 개별 수거 처리합니다.
- 모든 목표 오브젝트를 처리하면 해결 결과 화면이 나타납니다.

## 미션 판정 규칙

- 이벤트마다 고유한 성공 조건을 가집니다.
- 해양 쓰레기 이벤트는 모든 쓰레기 오브젝트가 수거함 영역에 드롭되면 해결됩니다.
- 미션 완료 전에는 이벤트 상태를 변경하지 않습니다.
- 점수 감점 여부는 초기 버전에서는 적용하지 않고, 난이도 확장 때 추가합니다.

## 업적 규칙

이벤트를 해결하면 업적이 추가됩니다. 업적은 지구 상태 화면과 이벤트 화면 모두에서 우측 상단부터 아래로 나열됩니다.

업적 예시:

| 업적 ID | 조건 | 표시 이름 |
| --- | --- | --- |
| first_rescue | 첫 이벤트 해결 | 첫 번째 회복 |
| fire_guardian | 산불 해결 | 숲의 수호자 |
| ocean_cleaner | 바다 쓰레기 해결 | 바다 청소자 |
| purifier | 오염 정화 이벤트 해결 | 정화의 손길 |
| perfect_run | 실패 이벤트 0개 | 완벽한 가이아 |

## 점수 계산 초안

최종 점수는 아래 요소를 합산합니다.

```txt
기본 점수 = 해결한 이벤트 수 * 100
위험도 보너스 = 노랑 해결 60 + 주황 해결 40 + 빨강 해결 20
시간 보너스 = max(0, 제한 시간 - 완료 시간 초) * 2
실패 패널티 = 실패 이벤트 수 * 50
업적 보너스 = 획득 업적 수 * 25

최종 점수 = 기본 점수 + 위험도 보너스 + 시간 보너스 + 업적 보너스 - 실패 패널티
```

점수는 0점 아래로 내려가지 않게 합니다.

## 게임 종료 조건

초기 버전에서는 `총 이벤트 수 10개`를 목표로 합니다.

- 해결 이벤트 수 + 실패 이벤트 수가 목표 이벤트 수와 같으면 게임 종료
- 이후 난이도 모드에서는 제한 시간, 이벤트 동시 발생 수, 위험도 상승 속도를 조정할 수 있습니다.
